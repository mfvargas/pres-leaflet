HTML

    El HTML (Hypertext Markup Language) es un lenguaje de marcas (tags) estándar para la creación de páginas y aplicaciones en la Web.

    Junto con CSS (Cascading Style Sheets) y JavaScript, conforma el grupo de las tres tecnologías principales de la Web.

    Fue creado en 1990 por el científico británico Tim Berners-Lee. La primera página html que hizo está aún disponible en http://info.cern.ch/.

HTML

    El HTML especifica la estructura y la semántica de una página web.

    La presentación es definida mediante CSS y la interactividad mediante el lenguaje JavaScript.

    Tanto HTML como CSS son estándares del World Wide Web Consortium (W3C).

HTML5

    Su primera especificación fue publicada por el W3C en 2014.

    Reduce la necesidad de complementos externos (ej. Flash) para el manejo de multimedia y otros recursos. Para esto, se apoya en CSS, JavaScript y en el Document Object Model (DOM).

    Se diseñó para funcionar en diferentes tipos de dispositivos (ej. computadoras, tabletas, teléfonos).

El Document Object Model

Ejemplo de documento HTML

    
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título del documento</title>     
</head>
<body>
    <!-- Contenido del documento -->
</body>
</html>

Elementos de HTML

    Son los componentes individuales de un documento HTML. Están compuestos por un tag inicial, contenido y un tag final.

Estructura de un elemento HTML


<tag atrib1="valor1" atrib2="valor2">contenido</tag>

    Algunos elementos pueden no tener tag final ni contenido.

    El valor de lo atributos se escribe siempre entre comillas dobles (""). Hay atributos particulares para cada elemento y también atributos globales.

Documentación sobre los elementos de HTML

Principales elementos HTML

    A continuación, se describen y se ejemplifican algunos de los principales elementos de HTML.

    En el título de cada diapositiva, se proporciona un enlace a la documentación del elemento en MDN.

DOCTYPE

    Informa al navegador la versión de HTML (o XML) en la que está escrita el documento.

    Es una declaración y no un elemento HTML.

    html es el valor correspondiente a HTML5.


        <!DOCTYPE html>

html

    Es el elemento raíz de un documento HTML.

    Contiene el resto de los elementos del documento.


        <!DOCTYPE html>
        <html>
          <head>...</head>
          <body>...</body>
        </html>

head

    Contiene los metadatos del documento y otros elementos como su título, scripts y hojas de estilo (CSS).


        <!DOCTYPE html>
        <html>
          <head>
            <meta charset="UTF-8">
            <title>Título del documento</title>
          </head>
        </html>

title

    Define el título del documento, el cual se muestra en la parte superior de la ventana o pestaña del navegador.

    Solo puede contener texto, los tags son ignorados.


        <title>Curso de programación en Python</title>

meta

    Especifica metadatos del documento, tales como la descripción, el autor o el juego de caracteres, entre otros.


        <head>
          <meta charset="UTF-8">
          <meta name="author" content="Manuel Vargas">
          <meta name="description" content="Curso introductorio de programación">
          <meta name="keywords" content="Python, programación">
          <title>Curso de programación en Python</title>
        </head>

body

    Especifica el contenido del documento (ej. texto, multimedia, hipervínculos).


        <!DOCTYPE html>
        <html>
          <head>
            <meta charset="UTF-8">
            <title>Curso de programación en Python</title>
          </head>
          <body>
            Python es un lenguaje de programación creado por Guido van Rossum...
          </body>
        </html>

h1 - h6

    Representan seis niveles de encabezados en el cuerpo del documento. h1 es el encabezado de mayor nivel (con letras más grandes) y h6 el de menor nivel.


        <body>
          <h1>El lenguaje Python</h1>
            <h2>Tipos de datos<h2>
              <h3>Numéricos</h3>
              <h3>Textuales</h3>
            <h2>Control de flujo<h2>
              <h3>Condicionales</h3>
              <h3>Ciclos</h3>
        </body>

p

    Se usa para representar un párrafo.

    El tag final es opcional.


        <body>
          <p>Python es un lenguaje de programación de propósito general ...</p>
          <p>Fue creado en 1989 por Guido van Rossum...</p>
        </body>re>

br

    Genera un salto de línea (o retorno de carro) en el documento.

    No debe usarse para incrementar el espacio entre los parrafos. Para eso se recomienda el elemento "p" o CSS.


        <p>
          Mozilla Foundation<br>
          1981 Landings Drive<br>
          Building K<br>
          Mountain View, CA 94043-0801<br>
          USA
        </p>

a

    Crea un hipervínculo a otro documento HTML, a un archivo, a una dirección de email o a cualquier otro tipo de URL (Uniform Resource Locator).

    El atributo href especifica el URL.


        <body>
          <a href="https://www.python.org/">El lenguaje Python</a>
        </body>

img

    Inserta una imagen en un documento. Este elemento se compone solo del tag inicial.

    El atributo src especifica el URL de la imagen y el atributo alt una descripción textual de la imagen.


        <img src="python-logo.jpg" alt="Logo de Python">

ul

    Define una lista no ordenada cuyos elementos se especifican mediante el elemento li (list item).


        Lista de compras:
        <ul>
          <li>Azúcar</li>
          <li>Harina</li>
          <li>Limones</li>
          <li>Arroz</li>
        </ul>

ol

    Define una lista ordenada cuyos elementos se especifican mediante el elemento li (list item).


        Lista de compras:
        <ol>
          <li>Azúcar</li>
          <li>Harina</li>
          <li>Limones</li>
          <li>Arroz</li>
        </ul>

table

    Se utiliza para presentar datos en filas y columnas.

    Los elementos thead y tbody especifican el encabezado y el cuerpo de la tabla, respectivamente.

    El elemento tr especifica las filas y td las columnas o las celdas. th se usa en las celdas de los encabezados.

table


        <table>
          <thead>
            <tr>
              <th colspan="2">Encabezado de la tabla</th>
            </tr>
          </thead>
          <tbody>
            <tr>
              <td>Fila 1,columna 1</td><td>Fila 1,columna 2</td>
            </tr>
            <tr>
              <td>Fila 2,columna 1</td><td>Fila 2,columna 2</td>
            </tr>
          </tbody>
        </table>

strong

    Denota la importancia de una parte del texto.

    Los navegadores acostumbran usar letra en negrita para implementar este elemento.


        <p>Esta parte de la oración es la <strong>más importante</strong>.</p>

em

    Se utiliza para denotar énfasis en una parte del texto.

    Los navegadores acostumbran usar letra en itálica para implementar este elemento.


        <p>Así se denota <em>énfasis</em> en una palabra.</p>

div

    Se utiliza para definir secciones o agrupar contenidos en un documento.


        <div style="background-color:#F00;">
          <h1>Sección roja</h1>
          <p>Esta sección tiene fondo rojo.</p>
        </div>
        <div style="background-color:#0F0;">
          <h1>Sección verde</h1>
          <p>Esta sección tiene fondo verde.</p>
        </div>
